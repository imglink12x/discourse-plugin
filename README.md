# ImgLink Discourse Plugin (Production)

This is an official ImgLink production integration package for **Discourse**.

## What this package includes
- Production integration hooks
- API upload helper
- Config example
- Reliability defaults (timeouts, structured error handling)

## Configure
1. Set API base URL (default: `https://imglink.cc`)
2. Set API key from your ImgLink dashboard
3. Enable plugin in your forum admin panel

## Notes
This is a production package intended for live deployment with per-forum-version validation.
